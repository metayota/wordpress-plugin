== Description ==

The Metayota Editor is perfectly integrated into WordPress! Utilize its unlimited possibilities to create a wide range of functionalities quickly and effortlessly.

**Features:**

- **Create Your Own Tags and Shortcodes:**  
  Utilize tags in HTML `<mytag parameter="value">` or with WordPress shortcodes `[guestbook]`.

- **Form Generator:**  
  Equip yourself with an array of components including dropdowns, checkboxes, radio boxes, etc.

- **Multilingual Support:**  
  Craft multilingual tags and forms and leverage automatic translation functions with ChatGPT and DeepL connections.

- **Database Editor:**  
  Create and edit your own database tables with components tailored to each data type, inclusive of search and sort functions.

- **Access Control:**  
  Define access levels to specific functions ensuring optimum security and functionality.

**Documentation and Support:**

- [Metayota Editor Documentation](https://www.metayota.com/en/documentation/): Comprehensive guide on tag creation with JavaScript and HTML, PHP web services building, and more.
- [WordPress Plugin Documentation](https://www.metayota.com/en/documentation-metayota-editor-wordpress-plugin/): A simplified overview of the WordPress plugin.
- [Video Tutorials on YouTube](https://www.youtube.com/@metayota): Visual, step-by-step guides.

== Installation ==

1. Download the Metayota Editor ZIP file.
2. Go to the 'Plugins' section in your WordPress admin panel and click 'Add New.'
3. Click ‘Upload Plugin’ and select the downloaded ZIP file, then click ‘Install Now.’
4. Activate the Metayota Editor through the 'Plugins' section.
